# String Reverse Program in Golang

Program ini berfungsi untuk membalikkan urutan sebuah string.

## Contoh Penggunaan

Input: `"hello world"`
Output: `"dlrow olleh"`

## Cara Menjalankan Program

1. Pastikan sudah menginstal Go di komputer.
2. Jalankan perintah berikut di terminal:
   ```bash
   go run main.go
   ```
